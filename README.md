## 什么是 Claude Code？

[Claude Code](https://docs.anthropic.com/en/docs/claude-code) 是 Anthropic 官方推出的 CLI 工具，让 Claude 直接在你的终端中协助编程。通过 `CLAUDE.md` 配置文件，可以定义项目规范、约束和最佳实践，让 AI 更好地理解项目上下文。
